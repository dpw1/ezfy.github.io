How does it work?

The development process is super simple!
Here is how it looks like:

You tell us your needs

We have a chat about what you want to implement or fix on your website. Once we talk it through we add all tasks into a Google Sheets file with their respective price and status.

We code it

A deadline is given and we start working. Once everything is finished we contact you, you double check whether everything is as you wish and if it's all good you send us the payment.

Enjoy your updated store!

That's it! On top of that we offer 7 days of free support in case any code we did breaks.
